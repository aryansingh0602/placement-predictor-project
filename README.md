# placement-predictor-project


About the Project
Many students feel confused about their chances of getting placed.  
This project is a simple attempt to predict whether a student might get placed based on a few basic factors.

---
Problem
Students often don’t have a clear idea of where they stand in terms of placement preparation.  
This can lead to stress and poor planning.

---

 Solution
This project uses a basic Machine Learning model to predict placement status using:
- CGPA  
- Study hours  
- Internship experience  

The goal is not perfect accuracy, but to understand how ML can be applied to real-life problems.



 Features
- Simple and easy-to-use interface  
- Quick prediction (Placed / Not Placed)  
- Built using beginner-friendly tools  



 Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- Streamlit
